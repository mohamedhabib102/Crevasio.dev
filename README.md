## üì¶ Getting Started

1. **Clone Repository:**
   ```bash
   git clone https://github.com/your-username/crevasio-dev.git
   cd crevasio-dev
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Run Development Server:**
   ```bash
   npm run dev
   ```

4. **Preview:**
   Open [http://localhost:3000](http://localhost:3000) in your browser.
